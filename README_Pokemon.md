# 🎮 Analyse Pokémon — Dashboard Power BI

> Projet de Business Intelligence | Power BI | Analyse de données Pokémon

---

## 📌 Description

Ce projet propose une **analyse interactive et visuelle** de l'univers Pokémon à travers un dashboard **Power BI**.  
Il explore les statistiques des Pokémon, leurs types, leurs talents, les dégâts, et les affrontements entre dresseurs.

---

## 🗂️ Structure du projet

```
📦 projet-pokemon-powerbi/
├── 📊 Projet_Pokemon.pbix                          # Dashboard Power BI principal
├── 📁 datapoke/
│   ├── ability_pokemon.csv                         # Capacités des Pokémon
│   ├── stats.csv                                   # Statistiques (PV, Attaque, Défense...)
│   ├── listetype.csv                               # Liste des types
│   ├── doubletype.csv                              # Pokémon à double type
│   ├── degattype1.csv                              # Dégâts par type (simple)
│   ├── degattypeA.csv                              # Dégâts par type (avancé)
│   ├── dresseur.csv                                # Données des dresseurs
│   └── talents_pokemon_note_ponderation.csv        # Pondération des talents
├── 🖼️  photo vs.png                                # Visuel comparatif
├── 🖼️  Capture d'écran console.png                 # Aperçu console
├── 🖼️  Capture d'écran logo pokemon.png            # Logo
└── 📋 README.md
```

---

## 📊 Contenu du Dashboard

- ✅ **Statistiques générales** — PV, Attaque, Défense, Vitesse par Pokémon
- ✅ **Analyse par type** — Forces et faiblesses, dégâts infligés/reçus
- ✅ **Double type** — Combinaisons de types et leurs avantages
- ✅ **Talents & capacités** — Classement et pondération des meilleurs talents
- ✅ **Dresseurs** — Comparaison et affrontements entre dresseurs

## 🚀 Ouvrir le projet

### Prérequis
- **Power BI Desktop** (gratuit) → [Télécharger ici](https://powerbi.microsoft.com/fr-fr/desktop/)

### Étapes
1. Cloner ou télécharger ce dépôt
2. Ouvrir **Power BI Desktop**
3. Fichier → **Ouvrir** → sélectionner `Projet_Pokemon.pbix`
4. Les données sont déjà intégrées ✅

---

## 📦 Sources de données

Toutes les données sont dans le dossier `/datapoke/` au format `.csv` :

| Fichier | Description |
|---|---|
| `stats.csv` | Stats complètes de chaque Pokémon |
| `ability_pokemon.csv` | Capacités disponibles par Pokémon |
| `listetype.csv` | Les 18 types Pokémon |
| `doubletype.csv` | Pokémon possédant deux types |
| `degattype1.csv` | Table des dégâts (type simple) |
| `degattypeA.csv` | Table des dégâts (type avancé) |
| `dresseur.csv` | Profils des dresseurs |
| `talents_pokemon_note_ponderation.csv` | Score pondéré des talents |
